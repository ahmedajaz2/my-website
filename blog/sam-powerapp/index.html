💻 Human-in-the-Loop Power App for Software Asset Management (SAM)

🗓️ Published: September 2, 2025 | ✍️ Author: Ajaz Ahmed

This project introduces a human-in-the-loop Power App designed to support Software Asset Management (SAM). It combines PowerApps, Copilot Studio, and Generative AI workflows to help SAM teams make informed software decisions while maintaining human oversight.

🧩 Overview

🏠 Homepage interface for navigating catalogs
📚 Catalog viewer with filtering, editing, and searching capabilities
🛠️ Python scripts to support AI-assisted pipelines for data analysis and preparation (coming soon)
💬 Planned Copilot Studio bot integration for conversational catalog search

🏠 Homepage

The homepage serves as the central command center, providing access to:

📦 Application Catalog
🚚 Supplier Catalog
📝 Contracts Catalog

📇 Application Catalog

Key features of the catalog include:

🔎 Search functionality to locate applications and suppliers efficiently
🧾 Filtered views based on approval status, license status, and review status
👁️ Details panel for viewing metadata of selected applications
✏️ Edit and delete options for catalog maintenance
🔁 Refresh function to synchronize with source data
🔙 Navigation controls to return to previous screens

🔮 Upcoming Enhancements

💬 Copilot Studio integration for conversational search
🤖 Generative AI pipeline scripts to suggest approvals, identify duplicates, and provide insights
📈 Power BI integration for catalog visualization and analytics

🛠️ Technology Stack

Microsoft PowerApps for application development
SharePoint or Dataverse for data storage
